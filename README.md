# MailChimp Newsletter Signup

 Newsletter signup using Node.js/Express and the Mailchimp API. 
 
 Please fill in your [audience ID] and your [API Key] in app.js

 
# Install dependencies
npm install
 
 # Serve on localhost:3000
npm start
npm run dev (Nodemon)
